# RAG-система: Стратегия развития ИИ в России до 2030

Система вопрос-ответ на базе **Retrieval-Augmented Generation**, которая отвечает строго по тексту официального документа - «Национальная стратегия развития искусственного интеллекта в Российской Федерации на период до 2030 года».

## Архитектура

```
PDF-документ
     │
     ▼
PyPDFLoader → Разбивка на чанки (RecursiveCharacterTextSplitter)
     │
     ▼
Эмбеддинги (multilingual-e5-base) → FAISS-индекс
     │
     ▼
Retriever (Top-K = 10) ──┐
                          ▼
                   LLM (step-3.5-flash via OpenRouter) → Ответ
```

## Стек технологий

| Компонент | Инструмент |
|---|---|
| Загрузка PDF | `PyPDFLoader` (LangChain) |
| Чанкинг | `RecursiveCharacterTextSplitter` |
| Эмбеддинги | `intfloat/multilingual-e5-base` (HuggingFace) |
| Векторное хранилище | `FAISS` |
| LLM | `nvidia/nemotron-3-super-120b-a12b:free` (OpenRouter) |
| Оркестрация | `LangChain` |
| Язык | Python 3.10+ |



Откройте `RAG_pipeline.ipynb` в Jupyter или Google Colab и запустите ячейки последовательно.

| `TOP_K` | 10 | Количество чанков в контексте |

## Принцип работы системы:

1. **Загрузка** - PDF разбивается на страницы с помощью `PyPDFLoader`
2. **Чанкинг** - текст делится на перекрывающиеся фрагменты для лучшего покрытия
3. **Индексирование** - каждый чанк кодируется многоязычной моделью `multilingual-e5-base` и сохраняется в FAISS
4. **Retrieval** - по заданному вопросу находятся Top-10 семантически близких чанков
5. **Генерация** - LLM формирует ответ строго по найденному контексту, без домыслов


## Оценка качества

Скрипт оценивает систему по четырём метрикам, вдохновлённым [RAGAS](https://docs.ragas.io):

| Метрика | Описание |
|---|---|
| **Faithfulness** | Ответ не содержит информации за пределами документа |
| **Answer Relevance** | Ответ соответствует заданному вопросу |
| **Refusal Accuracy** | Корректный отказ при prompt injection и отсутствии данных |

### Примеры ответов

**Фактический вопрос:**
> Что в Стратегии понимается под искусственным интеллектом?

> Искусственный интеллект определяется как комплекс технологических решений, позволяющий имитировать когнитивные функции человека (включая поиск решений без заранее заданного алгоритма) и получать при выполнении конкретных задач результаты, сопоставимые с результатами интеллектуальной деятельности человека или превосходящие их.

---

**Вопрос с фактической ошибкой (система корректирует):**
> Почему Стратегия устанавливает целевой объем услуг по разработке ИИ-решений в **12 млрд рублей** к 2030 году?

> Вопрос содержит ошибку: согласно Стратегии, ежегодный объём оказанных услуг в 2030 году должен вырасти не менее чем до **60 млрд рублей** (в 2022 году — 12 млрд рублей). Обоснование выбора этого целевого показателя в документе отсутствует.

---

**Вопрос об отсутствующих данных:**
> Какой минимальный оклад для ИИ-специалистов в госсекторе установлен Стратегией?

> Данная информация отсутствует в документе.
